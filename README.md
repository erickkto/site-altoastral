# Site Institucional — Espaço Alto Astral 🎈

Projeto prático desenvolvido para a disciplina de **Engenharia de Software** (TDE 03). O objetivo principal foi realizar o levantamento de requisitos com um cliente real e construir um site estático para fixar os conceitos de HTML5 semântico e estilização estruturada com CSS Grid Layout.

## 💻 Estrutura do Projeto

O site foi dividido de forma modular em 5 páginas para cobrir todas as necessidades do espaço:

* `index.html`: Home do site com banner principal e resumo dos diferenciais.
* `servicos.html`: Detalhamento dos ambientes (Salão de Festas, Área Kids, Brinquedoteca, Área de Jogos e Bar Retrô).
* `portfolio.html`: Galeria de fotos organizada para exibir a infraestrutura.
* `contato.html`: Central com endereço, integração de localização e links diretos para atendimento.
* `style.css`: Folha de estilo única responsável por toda a identidade visual e regras de layout do projeto.

---

## 🛠️ O que foi aplicado (Tecnologias)

* **HTML5 Semântico:** Uso correto de tags como `<nav>`, `<section>`, `<h2>` e `<h3>` para estruturação limpa do conteúdo, além de foco em acessibilidade com o atributo `alt` em todas as imagens.
* **CSS Grid Layout:** Alinhamento de elementos (como os cards da Home e as fotos da galeria) feito inteiramente através do modelo básico de Grid (`1fr`), sem códigos complexos ou automatizados.
* **Identidade Visual:** Paleta de cores personalizada baseada em tons terrosos (Marrom `#5c3d2e`, Bege `#c9a96e` e fundo claro `#fdfbf7`), focando no conforto de leitura e boa tipografia.
